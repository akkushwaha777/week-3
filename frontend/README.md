# Framboxx IT - Frontend

Welcome to the frontend application for **Framboxx IT**. This project is built using React and Vite to provide a fast, responsive user interface for our platform.

## Project Structure

- `src/` - Contains the main source code (components, styles, etc.)
- `public/` - Static assets
- `package.json` - Project dependencies and scripts

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at the local URL provided (usually `http://localhost:5173`).

## Technologies Used

- **React**: UI Library
- **Vite**: Build tool and dev server
- **Axios**: HTTP client for API requests
- **CSS**: Custom styling (responsive and modern)
